## InvoiceNo
Unique identifier for each order (invoice).

## StockCode
Unique product code assigned to each item.

## Description
Text description of the product.

## Quantity
Number of units purchased in a transaction.
Negative values indicate returns or cancellations.

## InvoiceDate
Date and time when the transaction occurred.

## UnitPrice
Price per unit of the product (in GBP).

## CustomerID
Unique identifier for a customer.
Some transactions have missing CustomerID values.

## Country
Country where the customer is located.
